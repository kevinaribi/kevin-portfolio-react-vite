# Kevin Portfolio Website — React + Vite V50.21

Conversion-safe React + Vite version based on V50.20, with Kairos premium finishing layer.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Original portfolio content, wording, assets, links, certificates, CV, QR, and selected works are preserved.
- V50.21 focuses on visual polish, responsive safety, smoother section reveal, and a cleaner Scope of Work / Selected Outputs presentation.
- This version is intended to replace the Bolt experiment as the stable React/Vite base.
